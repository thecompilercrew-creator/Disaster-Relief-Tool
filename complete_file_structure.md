1. **Sign Up**: https://cloud.mongodb.com (FREE)

2. **Create Cluster**:
   - Click "Build a Database"
   - Select "FREE" (M0 Sandbox)
   - Choose region closest to you
   - Click "Create"

3. **Create Database User**:
   - Go to "Database Access"
   - Click "Add New Database User"
   - Username: `admin`
   - Password: Click "Autogenerate Secure Password" (copy it!)
   - Database User Privileges: "Read and write to any database"
   - Click "Add User"

4. **Allow Network Access**:
   - Go to "Network Access"
   - Click "Add IP Address"
   - Click "Allow Access from Anywhere" (0.0.0.0/0)
   - Click "Confirm"

5. **Get Connection String**:
   - Go to "Database" â†’ Click "Connect"

   - Replace `<password>` with your actual password

## ğŸ› Common Issues & Solutions

### Issue: "npm: command not found"
**Solution**: Install Node.js from https://nodejs.org

### Issue: "Cannot find module 'express'"
**Solution**: 
```bash
cd backend
npm install
```

### Issue: "MongoNetworkError"
**Solution**: Check these in order:
1. Is MONGODB_URI correct in .env?
2. Did you replace `<password>` with actual password?
3. Is IP address whitelisted (0.0.0.0/0)?
4. Try pinging: `ping cluster0.xxxxx.mongodb.net`

### Issue: "Port 5000 already in use"
**Solution**:
```bash
# Mac/Linux: Find and kill process
lsof -i :5000
kill -9 <PID>

# Windows: Find and kill process
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# Or change PORT in .env to 5001
```

### Issue: Frontend shows "Connection error"
**Solution**:
1. Make sure backend is running (check terminal)
2. Check API_URL in index.html matches backend URL
3. Open browser console (F12) to see actual error
4. Clear browser cache: Ctrl+Shift+Delete

### Issue: "Token invalid" after login
**Solution**:
```javascript
// Open browser console (F12), type:
localStorage.clear();
// Then refresh and login again
```
